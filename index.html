Code Visualizers

  • Repl repl.it is a simple yet powerful online compiler, IDE, interpreter and interactive environment for programming languages. The name comes from the read-eval-print loop, the interactive toplevel used by languages like Lisp and Python
  • Pythontutor Despite its name, JavaScriptTutor helps people overcome a fundamental barrier to learning programming: understanding what happens as the computer runs each line of source code. Using this tool, you can write Python, Java, JavaScript, TypeScript, Ruby, C, and C++ code in your web browser and visualize what the computer is doing step-by-step as it runs your code. Over 3.5 million people in over 180 countries have used Python Tutor to visualize over 30 million pieces of code, often as a supplement to textbooks, lectures, and online tutorials.
  • JSFiddle JSFiddle was one of the earliest code playgrounds and a major influence for all which followed. Despite the name, it can be used for any combination of HTML, CSS and JavaScript testing.   • JSBin JS Bin is a tool for experimenting with web language (e.g., HTML, CSS, JavaScript, Markdown, Jade and Sass). Along with the code, the complete output of the code is also shared with other developers, colleagues and/or students. As you type into one of the editor "panels", you and anyone watching your bin will see the output being generated in real-time in the output panel.
